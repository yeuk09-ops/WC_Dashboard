# AI 분석 캐시

이 폴더는 OpenAI GPT를 통해 생성된 AI 분석 결과를 분기별로 저장합니다.

## 구조

각 분기별로 별도의 JSON 파일이 생성됩니다:
- `25-3Q.json` - 2025년 3분기 AI 분석 결과
- `25-4Q.json` - 2025년 4분기 AI 분석 결과
- ...

## 캐시 파일 형식

```json
{
  "quarter": "25.3Q",
  "generatedAt": "2026-01-09T12:00:00.000Z",
  "overview": "전체 개요 AI 분석 텍스트...",
  "turnover": {
    "국내(OC)": "국내 회전율 AI 분석 텍스트...",
    "중국": "중국 회전율 AI 분석 텍스트...",
    ...
  },
  "trend": {
    "국내(OC)": "국내 추세 AI 분석 텍스트...",
    "중국": "중국 추세 AI 분석 텍스트...",
    ...
  },
  "actionPlan": [
    {
      "priority": "HIGH",
      "label": "재고",
      "issue": "...",
      "action": "...",
      "target": "...",
      "responsible": "..."
    },
    ...
  ],
  "actionPlanInsight": "액션플랜 하단 AI 인사이트 텍스트..."
}
```

## 캐시 관리

- **자동 생성**: 데이터 업로드 또는 변경 시 자동으로 생성됩니다
- **재생성**: AI 재분석 버튼을 통해 수동으로 재생성할 수 있습니다
- **Git 포함**: 이 폴더는 Git에 포함되어 Vercel 배포 시 함께 배포됩니다

## 주의사항

- 이 캐시 파일들은 OpenAI API 호출 비용 절감을 위해 사용됩니다
- 데이터가 변경되지 않는 한 기존 캐시를 재사용합니다
- 분기별로 독립적으로 관리되어 과거 분기의 분석 결과는 보존됩니다
